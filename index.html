Projects

iOS Augmented Reality IlluminationSpace Tour

SUMMER 2023

This project is an iOS application designed to support STEM education at the City University of New York (CUNY) Advanced Science Research Center (ASRC). It uses Apple's RealityKit to run multiple Augmented Reality (AR) scenes in a scavenger hunt around the ASRC.

This AR application was designed as a part of the 2023 VR-REU run by Professor Wole. As the name suggests, the summer program is an REU, i.e. Research Experience for Undergraduates, sponsored by the National Science Foundation (NSF). This project specifically was supervised by Kendra Krueger.

To learn more about how the application works, please visit the GitHub repository and look at the ReadMe: GitHub Link
